Basic Exercise: const PI = 3.14;

  1. What's the difference between var and let?
     Unlike Var, Let can't be redefined or hoisted, is block scoped (not function scoped), and won't impact window's object.

  2. What's the difference between var & const?
    Same as differences between var & let, but const also can't be reassigned. Const must also be initialized when declared

  3. What's the difference between let & const?
    Const must be initialzied when declared & can't be reassigned

  4. What is hoisting?
    Hoisting is the ability to utilize a variable before it's declared.
